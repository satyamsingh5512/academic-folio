# Satyam Singh

Personal portfolio built with Next.js and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Visit `localhost:3000` to view locally.

## Build

```bash
npm run build
npm start
```

## Stack

Next.js 16, TypeScript, Tailwind CSS

## Deploy

Works on Vercel, Netlify, or any Next.js-compatible platform.
